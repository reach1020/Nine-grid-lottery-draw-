# 九宫格抽奖程序
🎉 基于HTML+CSS+JS实现的前端九宫格抽奖demo

[![访问网站](https://img.shields.io/badge/访问网站-点击进入-blue)](https://reach1020.github.io/Nine-grid-lottery-draw-/)

## 技术栈
- HTML5
- CSS3 Grid 布局（实现九宫格）
- JavaScript 递归+setTimeout（实现滚动减速逻辑）

## 快速运行
1. 下载仓库所有文件（HTML/CSS/JS）；
2. 直接用浏览器打开`lottery.html`文件，点击中间按钮即可抽奖。

## 核心功能
- 防重复点击：抽奖过程中按钮失效；
- 动态减速：滚动后期逐渐变慢，模拟真实转盘；
- 结果展示：抽中奖品/谢谢参与，顶部弹窗提示；
- 固定布局：视窗缩放不改变元素尺寸。
